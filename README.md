# 📚 BookJett

A modern online bookstore website built using HTML, CSS, and JavaScript.

## 🚀 About

BookJett is a personal learning project where I'm building a complete bookstore website from scratch while improving my frontend development skills.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
